给定一个大小为 n 的整数数组，找出其中所有出现超过 ⌊ n/3 ⌋ 次的元素。

进阶：尝试设计时间复杂度为 O(n)、空间复杂度为 O(1)的算法解决此问题。

**示例 1：**:<br>
```
输入：[3,2,3]
输出：[3]
```

**示例 2：**:<br>

```
输入：nums = [1]
输出：[1]
```

**示例 3：**:<br>

```
输入：[1,1,1,3,3,2,2,2]
输出：[1,2]
```

**提示：** <br>
* 1 <= nums.length <= 5 * 104
* -109 <= nums[i] <= 109
