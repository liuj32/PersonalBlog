给定一个整型数组，在数组中找出由三个数组成的最大乘积，并输出这个乘积。

**示例 1：**:<br>
```
输入: [1,2,3]
输出: 6
```

**示例 2：**:<br>

```
输入: [1,2,3,4]
输出: 24
```

**注意：** <br>
1. 给定的整型数组长度范围是[3,10^4]，数组中所有的元素范围是[-1000, 1000]。<br>
2. 输入的数组中任意三个数的乘积不会超出32位有符号整数的范围。<br>
