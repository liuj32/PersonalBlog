给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（格式正确且连续）括号子串的长度。v

**示例 1：**:<br>
```
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
```

**示例 2：**:<br>

```
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
```

**示例 3：**:<br>

```
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
```

**提示：** <br>
* 0 <= s.length <= 3 * 10^4
* s[i] 为 '(' 或 ')'
