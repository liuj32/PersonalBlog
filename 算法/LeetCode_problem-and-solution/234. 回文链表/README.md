请判断一个链表是否为回文链表。

**示例 1：**:<br>
```
输入: 1->2
输出: false
```

**示例 2：**:<br>

```
输入: 1->2->2->1
输出: true
```
**提示：** <br>
* 你能否用 O(n) 时间复杂度和 O(1) 空间复杂度解决此题？
