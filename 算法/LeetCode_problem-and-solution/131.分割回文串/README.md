给定一个字符串 s，将 s 分割成一些子串，使每个子串都是回文串。

返回 s 所有可能的分割方案。

**示例 1：**:<br>
```
输入: "aab"
输出:
[
  ["aa","b"],
  ["a","a","b"]
]
```
