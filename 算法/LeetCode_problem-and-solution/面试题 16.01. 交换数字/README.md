编写一个函数，不用临时变量，直接交换numbers = [a, b]中a与b的值。

**子数组** 定义为原数组中的一个连续子序列。<br>
请你返回 arr 中 **所有奇数长度子数组的和 。**

**示例 1：**:<br>
```
输入: numbers = [1,2]
输出: [2,1]
```

**提示：** <br>
* numbers.length == 2
