给你一个整数 n ，请你判断 n 是否为 丑数 。如果是，返回 true ；否则，返回 false 。

丑数 就是只包含质因数 2、3 和/或 5 的正整数。

**示例 1：**:<br>
```
输入：n = 14
输出：false
解释：14 不是丑数，因为它包含了另外一个质因数 7 。
```

**示例 2：**:<br>

```
输入：n = 6
输出：true
解释：6 = 2 × 3
```

**示例 3：**:<br>

```
输入：n = 8
输出：true
解释：8 = 2 × 2 × 2
```

**提示：** <br>
* n == nums.length
* 1 <= n <= 104
* 0 <= nums[i] <= n
* nums 中的所有数字都 独一无二
