# 浏览器

## 浏览器如何渲染网页
1. 处理 HTML 并构建 DOM 树。(建立起父亲、儿子、兄弟之间的关系)
2. 处理 CSS构建 CSSOM 树。
3. 将 DOM 与 CSSOM 合并成一个渲染树。
4. 根据渲染树来布局，计算每个节点的位置。
5. 调用 GPU 绘制，合成图层，显示在屏幕上(栅格化：合成线程会按照视口附近的图块来优先生成位图->浏览器进程根据指令生成页面，显示在屏幕上)
